# NodeAuth

Usage
----
Create .env file
    DB_URL='{Mongodb-connection-url}'
    PORT={Port-number}
    URL='{URL-with-port}'
    GCLIENT_ID='{Google-API-client-id}'
    GSECRET='{Google-API-Key}'
    SECRET='{Any-secret-key}'

RUN
---
    Current-directory# npm install
        some processing
    Current-directory# node app.js
