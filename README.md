# NodeAuth

Usage
----
Create .env file
    DB_URL='{Mongodb-connection-url}'<br>
    PORT={Port-number}<br>
    URL='{URL-with-port}'<br>
    GCLIENT_ID='{Google-API-client-id}'<br>
    GSECRET='{Google-API-Key}'<br>
    SECRET='{Any-secret-key}'<br>

RUN
---
    Current-directory# npm install
        some processing
    Current-directory# node app.js
